
DESCRIPCION
-----------

El módulo dge_widget es un modulo desarrollado a medida que implementa un bloque mediante el cual se permite crear widgets HTML para mostrar en otras webs. 


REQUISITOS
----------

Requiere de una instalación previa del portal Datos.gob.es (Drupal 7)




INSTALACION
-----------

- Copiar la carpeta con el módulo en sites/all/modules/
- Activar el módulo en la administración de módulos (Sección Catálogo, módulo dge_widget)
