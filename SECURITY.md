# Seguridad

## Reporte de vulnerabilidades
Si encuentras una vulnerabilidad de seguridad:
- Evita abrir un issue público con detalles explotables.
- Contacta con el equipo mantenedor a través de los [canales internos de la plataforma](https://datos.gob.es/contact) o, si aplica, mediante el mecanismo de reporte de seguridad de GitHub del repositorio afectado.

## Alcance
Este repositorio es principalmente documentación. Las vulnerabilidades normalmente afectarán a:
- Extensiones CKAN (repositorios `ckanext-*`).
- Drupal (repositorios de módulos/temas).
